# sysadmin-scripts

A collection of one-liner shell commands (Bash, AWK, grep, find) used for system administration, process management, and text processing.

## Categories
* **File Management:** Searching, permission modification, size verification (`find`, `chmod`, `stat`).
* **Process & System:** Memory monitoring, resource usage, parent-child relations (`ps`, `kill`).
* **Data Processing:** Filtering and formatting system files output (`awk`, `grep`, `cut`, `uniq`).

## Usage
Ready-to-use one-liners for quick terminal execution.

## Functional Scripts
More complex, reusable Bash scripts for automated tasks, log parsing, and system monitoring.
